# Algorithm

## Orbit Propagation
- 2-body model
- RK4 적용

### 2-body model
두 개의 질량 (지구, 위성)만 고려해서 중력만으로 운동을 계산하는 모델

#### 컨셉
현실에 존재하는 다양한 외력과 변수를 줄여 두 물체간의 상호작용만 고려하여 계산하는 방법
2-body model은 다음과 같은 가정하에 계산한다.
1. 지구는 완벽한 구형
2. 중력만 작용
3. 다른 외력은 없다
4. 질량 중심 기준으로 운동

#### 특징
- 기본 궤도 표현 가능
- 지구 중심 방향으로 끌림
- 지구 중심에서 멀어질수록 약해짐
- 구현이 쉬움

#### 수식
##### 위성 운동의 기본 방정식
a = (- (지구 중력 상수) / (r * r * r)) * r
- a : 가속도
- r : 위치 벡터
- r * r * r : 거리^3
- 지구 중력 상수 : (중력 상수) * (지구 질량)


### RK4 (Runge-Kutta)
4차항까지 구하여 사용하는 수치적분 방법

#### 컨셉
dt = 0, dt = h/2, dt = h 세 지점의 기울기 평균을 통해 다음 step의 값을 계산하는 방법

#### 특징
- 높은 정확도
- 구현이 간단함

#### 수식
(y_i + 1) = (y_i) + 1/6(k_1 + 2 * k_2 + 2 * k_3 + k_4) * h
- k_1 = f(t_i, y_i)
- k_2 = f(t_i + 1/2 * h, y_i + 1/2 * k_1 * h)
- k_3 = f(t_i + 1/2 * h, y_i + 1/2 * k_2 * h)
- k_4 = f(t_i + h, y_i + k_3 * h)
